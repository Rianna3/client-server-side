Introduction

Client side

By clicking here, you can find the content about great inventions and inventors.

Server-side

URL Function
/inventions All the data about inventions.
/inventors All the data about inventors
/inventions/add Add extra data about invention.
/inventors/add Add extra data about inventor
/inventions/:id (e.g. /inventions/6) Find a specific invention data.
/inventors/:id (e.g. /inventors/2) Find a specific inventor data.
